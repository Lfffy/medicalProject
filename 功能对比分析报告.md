# 医疗疾病数据分析大屏可视化系统 - 功能对比分析报告

## 📊 系统概述

这是一个基于Vue.js前端和Flask后端的医疗疾病数据分析大屏可视化系统，主要包含普通医疗数据管理和孕产妇健康管理两大模块。

## 🎯 前端已实现功能

### 1. 核心页面组件
- **Index.vue** - 主页面可视化大屏
  - 各年龄段患病占比/各孕周分布图表
  - 疾病类型分布/风险等级分布
  - 病例列表/孕产妇信息列表
  - 医院科室环形图/风险等级分析
  - 疾病关键词云图/健康指标分析
  - 孕期健康指标追踪图表

- **DataCenter.vue** - 数据中心
  - 数据管理（列表展示、筛选、分页）
  - 数据录入（医疗数据、孕产妇数据表单）
  - 数据导入（文件上传功能）
  - 数据表格展示

- **AnalysisCenter.vue** - 智能分析预测中心
  - 智能预测（疾病风险、孕产妇风险、健康趋势）
  - 对比分析（不同数据类型对比）
  - 趋势分析（时间序列分析）
  - 风险评估（特征重要性分析）

- **MonitoringCenter.vue** - 监测中心
- **DashboardCenter.vue** - 可视化大屏
- **Auth.vue** - 认证中心

### 2. 辅助功能页面
- **TrendAnalysis.vue** - 趋势分析
- **ComparisonAnalysis.vue** - 对比分析
- **PredictionModel.vue** - 预测模型
- **ReportGeneration.vue** - 报告生成
- **PrenatalCare.vue** - 产检记录
- **FetalMonitoring.vue** - 胎心监测
- **NutritionAdvice.vue** - 营养建议
- **WarningSystem.vue** - 预警系统

### 3. 基础架构组件
- **UnifiedLayout.vue** - 统一布局组件
- **Header.vue** - 头部导航
- **NavigationMenu.vue** - 侧边栏导航
- **StarryBackground.vue** - 星空背景效果

### 4. 工具和配置
- **axios.js** - HTTP请求配置
- **router/index.js** - 路由配置
- **store/index.js** - 状态管理

## 🔧 后端已实现功能

### 1. 核心API模块

#### data_management_api.py - 数据管理API
- `GET /api/data/list` - 获取数据列表（支持分页、筛选）
- `POST /api/data/medical/add` - 添加医疗数据
- `POST /api/data/maternal/add` - 添加孕产妇数据
- 数据导入导出功能
- 文件上传处理（支持xlsx、xls、csv）

#### analysis_api.py - 分析预测API
- `GET /api/analysis/trend` - 趋势分析数据
- 智能预测算法
- 对比分析功能
- 风险评估模型
- 报告生成功能

#### maternal_api.py - 孕产妇专项API
- `GET /api/maternal/records` - 获取孕产妇记录列表
- `GET /api/maternal/record/<id>` - 获取单个记录详情
- `POST /api/maternal/record` - 创建孕产妇记录
- `PUT /api/maternal/record/<id>` - 更新孕产妇记录
- `DELETE /api/maternal/record/<id>` - 删除孕产妇记录

#### user_api.py - 用户管理API
- `POST /api/user/register` - 用户注册
- `POST /api/user/login` - 用户登录
- `GET /api/user/profile` - 获取用户信息
- `PUT /api/user/profile` - 更新用户信息
- 密码哈希和验证

#### hospital_api.py - 医院管理API
- `GET /api/hospitals` - 获取医院列表
- `GET /api/hospitals/<id>` - 获取医院详情
- `POST /api/hospitals` - 添加医院
- `PUT /api/hospitals/<id>` - 更新医院信息
- `DELETE /api/hospitals/<id>` - 删除医院

#### permission_api.py - 权限管理API
- 权限分配和管理
- 角色权限控制
- 操作权限验证

#### log_api.py - 操作日志API
- 用户操作记录
- 日志查询和统计
- 安全审计功能

### 2. 数据库设计

- **maternal_info** - 孕产妇信息表
